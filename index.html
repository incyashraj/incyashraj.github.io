01 ·

so, layer36 — in pieces.

↑ pin it to your fridge
step 01 · source

your app.

written once, compiled to WebAssembly. no six codebases, no platform glue.

step 02 · surface

universal api.

a small, opinionated call surface. files, processes, network — one dialect.

step 03 · check

UCap permissions.

every call gets a permission check. default-deny.

step 04 · host

native adapters.

approved calls translate to the OS underneath. same .wasm, anywhere it lands.

~ the whole point ~

software, portable like files.

You don't think about a PDF working on Mac and Windows. You just send it. Layer36 wants apps to feel like that — write once, capability-checked, ride the host adapter wherever you need to land.

currently shipping v0.1.0-rc1
what works today

v0.1.0-rc1.

  • desktop runtime (rust + wasmtime)
  • CLI: run · version · doctor
  • UAPI for CLI-style apps
  • UCap capability checks
  • same .wasm on win / mac / linux
  • threat model · ADRs · public docs
