01 ——

Layer36.

The thing I'm building

Layer36 lets apps compile to WebAssembly and call a common Universal API instead of platform-specific APIs. The runtime checks permissions through UCap, then host adapters translate approved calls to the native operating system.

Built in Rust on Wasmtime. Working CLI (run, version, doctor), CI running the same .wasm binary across three desktop platforms, a full threat model, benchmarks, ADRs, and public docs. Currently at v0.1.0-rc1.

The long-term goal: one app model across Windows, macOS, Linux, Android, iOS, ChromeOS, and the web.

Today · v0.1.0-rc1

  • Working desktop runtime (Rust + Wasmtime)
  • CLI: run · version · doctor
  • Early UAPI coverage for CLI-style apps
  • Capability checks via UCap (default-deny)
  • CI runs same .wasm on Win / macOS / Linux
  • Threat model, benchmarks, ADRs, public docs
